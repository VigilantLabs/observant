# Vigilant Front-end
[![MIT License](http://b.repl.ca/v1/License-MIT-red.png)](LICENSE)

Front-end Webapp for Vigilant

##Setup

Flask Angularjs Dashbaord

```bash
$ pip3 install requirements.txt
$ bower install

# development
$ ./dashboard.py

# optional nginx development
$ ./nginx.sh
```
